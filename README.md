## 🧠 Projets phares

### Mackandal AI

**Mackandal AI** est un projet open source ambitieux visant à créer une **intelligence linguistique** adaptée au **créole haïtien** et autres langues marginalisées. Il repose sur des technologies modernes et éthiques, pour rendre l’IA accessible à tous.

#### 🎯 Objectifs :
- Traduction automatique FR ↔ Kreyòl (`Tradiktè Mackandal`)
- Génération assistée de CV et lettres de motivation (`Resous Travay`)
- Assistant local intelligent pour démarches et informations quotidiennes (`Info Lokal`)

#### 🧰 Technologies utilisées :
- **FastAPI** – API REST performante en Python
- **Hugging Face** – NLP et modèles LLM personnalisés
- **Prompt Engineering** – Construction de prompts immersifs et optimisés
- **Git / GitHub** – Collaboration, versionnage, gestion de projet

➡️ [Voir le projet Mackandal](https://github.com/FalandyJEAN/mackandal)

### SnapTrad

**SnapTrad** est une application de **trading algorithmique** que j'ai développée récemment. Elle utilise des stratégies basées sur l'analyse technique pour détecter des signaux de trading en temps réel.

#### 🧰 Technologies utilisées :
- **Python** – Développement de l'algorithme de trading
- **Node.js / Express** – Backend pour la gestion des requêtes
- **JavaScript** – Interface utilisateur interactive
- **Git / GitHub** – Versionnage et collaboration

➡️ [Voir le projet SnapTrad](https://github.com/FalandyJEAN/snaptrade)
