### 🇭🇹 Fintech & SDKs

**[bazik-sdk](https://www.google.com/search?q=https://pypi.org/project/bazik-sdk/)**
Async Python client for the Bazik API — MonCash & NatCash payments, payouts, transfers and wallet for Haiti 🇭🇹. Fully typed, one dependency (`httpx`), production-tested.

```bash
pip install bazik-sdk

```

**[fabrik-fastapi](https://pypi.org/project/fabrik-fastapi/)**
Opinionated FastAPI project generator (CLI). One command scaffolds JWT auth, an auto-discovered async admin UI, ARQ background tasks, Alembic migrations and isolated tests — production-ready in 60 seconds.

```bash
pip install fabrik-fastapi

```
